Experiment 4 - Infinite Worlds(Farmworld)

Description

It creates a generative infinite farmworld based in Davis, California

Technical

The designs of the fields, roads, and river uses a hash function to determine its size and placement. Additionally, the river uses a perlin noise function to determine how the river would be designed. The river is animated using random function to create a shimmer-like effect, and pressing the tiles would generate long grass or wheats.

Reflection

It was simple to create the fields and the river using random and hash functions to determine its placements. The user input interactions was a bit more challenging with many test to achieve the grass look and what could be interactable or not.
