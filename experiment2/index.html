Experiment 2 - Living Impressions

Description

This creates a generative landscape, inspired by an image in Tianchi, a lake in China

Technical

The sky and the lakes are designed using a rectangle. The mountains, hills, and trees are generated uses a noise function and offsets to make the terrains. The clouds and lake shimmer uses an array and random() to set up the location, size, and speed of the eclipses.

Reflection

The difficults were trying to make the terrains look smoother and nicer, in which using the p5.js noise examples to figure out how to make it look more appealing.
