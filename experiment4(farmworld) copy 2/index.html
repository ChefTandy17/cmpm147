Experiment 4 - Infinite Worlds(Artic Islands)

Description

It creates a generative infinite artic islands based in Aleutian Islands, Alaksa

Technical

The designs of the snow and permafrost uses a hash function to determine its size and placement. The ocean uses a noise function and is placed on the top layer to make it look like there are a bunch of islands. The ocean is animated using random to create a shimmer-like effect, and pressing the tiles will create small grasses.

Reflection

The ocean noise function was discovered accidentally when trying to make the islands. The user input interactions was a bit more challenging with many test to achieve the grass look and what could be interactable or not.
