Experiment 4 - Infinite Worlds(Desert Farmworld)

Description

It creates a generative infinite desert farmworld based in Phoenix, Arizona

Technical

The designs of the fields, roads, and canal uses a hash function to determine its size and placement. Additionally, the canal uses a perlin noise function to determine how the river would be designed. The canal is animated using random function to create a shimmer-like effect, and pressing the tiles would generate grass. Also, the fields could be rectangles, circles, or half circles.

Reflection

It was simple to create the fields and the canal using random and hash functions to determine its placements. The user input interactions was a bit more challenging with many test to achieve the grass look and what could be interactable or not. The circular fields took a bit of time to figure out and debugged, as it sometimes would mess up the worlds appearance.
