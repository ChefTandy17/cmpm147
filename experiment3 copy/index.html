Experiment 3 - Alternate Worlds (Dungeon)


Description

Randomly generates a different dunegon when choosing a different seed

Technical

It uses randomness() when it comes with adjusting the width and height and chest placements, placing the hallways, and the amount of rooms that would be in the dungeon

Reflection

For the dungeon world, creating randomness for the width and height of the dungeons, and placements weren't too challenging. Connecting hallways required a lot of researching and generative AI to fully understand the basics and the structure of a generative dungeon
