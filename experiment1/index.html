Experiment 1 - Generative Methods/Getting Started

Description

This experiment produces a very short generated story

Technical

The "generate" function goes through each word in the template to find a placeholder word. When a placeholder word is detected, it would use the placeholder word to find if it matches the names of the arrays provided to start randomizing and reinserted to remove the placeholder

Reflection

It was easy to observe how the generative stories are created in Glitch when it comes with the specific words are being replaced or are placeholders in the story. It was harder to understand the functions of the replacer, generator, and how a slot pattern works that was provided for us, but eventually I learned that it uses while loops, if-else statements, and psudo-randomness to create a simple generated story.
