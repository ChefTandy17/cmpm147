Experiment 1 - Generative Methods/Getting Started

Description

This experiment produces a very short story in which when refreshing the page would create random locations and names?

Technical

The generate functions uses one of the variables in the arrays to select a random string along with finding a placeholder. The replacer function replaces placeholders in the template and insert the randomly selected string. The process continues until there isn't any avaliable words in the story

Reflection

It was easy to observe how the generative stories are created in Glitch when it comes with the specific words are being replaced or are placeholders in the story. It was harder to understand the functions of the replacer, generator, and how a slot pattern works.
